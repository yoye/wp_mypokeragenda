wp_mypokeragenda by yoye

Plugin wordpress vous permettant d'ajouter des tournois de poker poker dans vos posts ou vos pages.

Description

Ce plugin vous permet d'ajouter facilement des listes de tournois sur vos pages Wordpress. Bénéficiez des données stockées par mypokeragenda pour créer facilement du contenu sur votre site.

Vous bénéficiez ainsi d'une base de données de 200 casinos, des salles de poker en ligne agréées par l'ARJEL et également de la liste de tous les principaux tournois internationaux.

Installation

Téléchargez le plugin à l'aide de ce lien : télécharger. Décompressez le contenu et renommez le en "mypokeragenda", placez maintenant le dossier avec vos plugins wordpress dans : "/wp-content/plugins". Ensuite il ne vous reste plus qu'à activer le plugin.

Configuration

Une fois que le plugin est activé allez dans le menu "Réglages" de votre espace d'administration wordpress et cliquez sur "mypokeragenda". Vous pouvez à présent configurer les widgets que vous souhaitez mettre en place sur vos posts ou vos pages.

Cherchez parmis les listes disponibles par exemples : "European Poker Tour", "Winamax" et sélectionné le nombre de résultats que vous souhaitez afficher. Une fois votre configuration enregistrée elle apparaîtra dans la liste en dessous du formulaire, il vous suffira de copier le code fournis qui doit ressembler à cela: 

[mypokeragenda id="1"]

Téléchargement

Vous pouvez télécharger ce plugin au format zip ou tar.

Vous pouvez également cloner ce projet avec Git grâce à la commande: 

$ git clone git://github.com/yoye/wp_mypokeragenda
